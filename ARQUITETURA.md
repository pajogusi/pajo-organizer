# Pàjó Organizer — Arquitetura v1.0

Projeto modular em HTML, CSS e JavaScript.

## Módulos

- Dashboard
- Agenda
- Tarefas
- Notas
- Contactos
- Projetos
- Veículos
- Combustível
- Finanças
- Viagens
- Mundo
- Biblioteca
- Documentos
- Definições

## Regra de desenvolvimento

Um módulo de cada vez.

Cada módulo pode ter:

- index.html
- style.css
- script.js

## Objetivo

Criar um organizer digital inspirado nos organizers clássicos, agendas em papel e sistemas modernos tipo Notion, mas simples, offline e sem dependência de serviços externos.
